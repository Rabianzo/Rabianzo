Java Dev
--------

Welcome to my GitHub—a digital workshop where code gets built, debugged, and occasionally wrestled into submission. I speak HTML, CSS, and JavaScript—the essentials for making the web work. TypeScript also gets an invite because, well, strict typing saves lives (and my sanity). Dart and Flutter let me build apps that work on any device—because, honestly, who has the time for platform-specific headaches? Then there’s Python, my reliable buddy for scripting and automation. It’s like the Swiss Army knife of programming languages, and it’s got my back on all kinds of projects. And, yeah, I dabble in C. It’s a gritty language, and working close to the hardware feels like the code version of bench-pressing—tough, but rewarding. Outside of coding, I mess around with design and keep the ideas flowing. I’m always up for learning something new, whether it’s a slick new framework or a better way to wrestle code into submission. Take a look around, star a repo if it catches your eye, and don’t be afraid to fork (code, not the other kind). Let's make something that actually works—or at least looks like it does! Happy coding, and may your bugs be few. 🧰
